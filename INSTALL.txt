3. Installation

  For Mac OS:

  1. Install XQuartz https://www.xquartz.org/
  2. Run 'make install-deps' to install the required dependencies
  3. If a version of GLUT isn't already installed, use Homebrew to install FreeGLUT with 'brew install freeglut'
  4. 'make build' to build the executable
  5. 'make start' to start the game
